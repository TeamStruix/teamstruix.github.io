Sample RAW files

For App Review — try the computational merge without a physical camera.

Reviewer note: A full walkthrough video is attached to this submission. If you would like to test hands-on, these five RAW frames form one burst of the same scene and can be merged in the app.

How to use them

  1. On the test iPhone, download the ZIP below in Safari and open it — tap Save to Files.
  2. In the Files app, tap the ZIP to extract it, then select all five .ORF files → Share → Save Images to add them to Photos.
  3. Open Struix, go to the Tasks tab, and tap Import RAW. Select all five frames.
  4. They import as one burst on the Library tab. Open the set and tap Merge to run the on-device computational merge, then view the before/after compare.
  5. To test conversion instead, pick a single frame and use Convert to produce a DNG.
